# Universo Infinito 🌌❤️

Una experiencia web inmersiva en 3D creada con **Three.js**. Presenta un corazón de partículas mágico con efectos de aura, destellos cósmicos y cometas en movimiento.

## Características
- **Corazón Paramétrico:** Generado con ecuaciones matemáticas y animado con shaders GLSL.
- **Efectos Visuales:** Bloom (resplandor), Aura mágica, Destellos (sparkles) y Cometas.
- **Interactividad:** El universo reacciona al movimiento del ratón y es totalmente responsive.
- **Diseño Premium:** Tipografía elegante y paleta de colores galáctica.

## Cómo usar
Simplemente abre `index.html` en cualquier navegador moderno.

## Tecnologías
- HTML5 / CSS3
- JavaScript (ES6+)
- [Three.js r128](https://threejs.org/)
